# fe-utils

用 ts 写的前端工具库，持续更新

| File            | Use                                       | Async |
| --------------- | ----------------------------------------- | ----- |
| url-to-blob     | url 转 blob 对象，支持 dataUrl 和 blobUrl | True  |
| load-image      | 加载图片                                  | True  |
| parse-url-query | 解析 url 的 query                         |       |
| deep-copy       | 深拷贝                                    |       |
| get-ext         | 获取 string 或 FileLike 对象后缀名        |       |
| type-of         | 检查一个值的类型                          |       |
| delay           | 等待一段时间后执行                        | True  |
| select-file     | 选择文件                                  | True  |
